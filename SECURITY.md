# Security Policy

BudgetPilot is currently a self-hosted household/LAN application, not a
public internet service. Read [docs/SECURITY.md](docs/SECURITY.md) before
deploying it.

Please report security issues through GitHub Private Vulnerability Reporting
when it is available for this repository.

If no private reporting channel is configured, open a minimal public GitHub
issue asking for a private contact path. Do not include exploit details,
real financial records, passwords, receipt photos, private network details,
local paths, screenshots, or data-file contents in a public issue.
